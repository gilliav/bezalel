01 Design Principles

01

Warm & Tactile

The parchment background and grid texture evoke physical sketchbooks and studio notebooks. Every screen should feel like a working surface, not a dashboard.

02

Color as Identity

Each course and assignment gets a distinct pastel hue. Color is the primary recognition system — not labels or icons. Keep palette muted and unpredictable.

03

Urgency at a Glance

Timer badges (2d, 1w) communicate deadline pressure in a single token. The design never buries urgency — always visible, never alarming.

04

Typography as Character

Almoni is used exclusively — a rounded, confident Hebrew typeface. Bold weight for titles, Medium for supporting info. Never default to system fonts.

05

RTL Native

All layouts are designed right-to-left. Text anchors right, back navigation sits top-left, and visual hierarchy flows in the natural Hebrew reading direction.

06

Structured Simplicity

The 24px grid and section banner system create strong visual rhythm. Information is layered in clear zones — no decorative complexity, just purposeful structure.

04 Spacing & Grid

Base Grid Unit

24px

All spacing values are multiples or fractions of 24px. Grid lines are visible as a subtle background texture.

Screen width & Gutter

375px

Standard iOS mobile viewport. Horizontal gutter is 26px (≈ 24px grid + 2px). Column content is ~323px wide.
